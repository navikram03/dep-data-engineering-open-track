# Weekly Fuel Price Volatility and Spike Detection in Cavite

## Problem Statement
I want to answer: 
"How volatile are weekly gasoline and diesel prices in Cavite during 2026?"

## Audience
This project is for daily commuters in Cavite.

## KPI or Key Metric
The main metric I want to track is fuel price volatility.

## Likely Data Source
I will explore Department of Energy (DOE) Philippines (https://www.doe.gov.ph/)

## Possible Final Dashboard
The dashboard should help the audience quickly see fuel price trends, volatility, and spike occurrences in Cavite during 2026.
